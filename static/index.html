REST API Documentation

이 서비스는 REST API를 통해 iOS, Android, PC 등 다양한 플랫폼에서 연동할 수 있습니다.

API Endpoints

GET /api/transform/styles

사용 가능한 캐릭터 스타일 목록을 조회합니다.

POST /api/transform/character

인물 사진을 캐릭터 이미지로 변환합니다.

Parameters (multipart/form-data):
  • image - 인물 이미지 파일 (필수)
  • style - 스타일 ID (기본값: sd_character)
  • denoising_strength - 변환 강도 0.3~0.9 (추천값: 0.42)
GET /api/transform/image/{image_id}

생성된 캐릭터 이미지를 조회합니다.

GET /api/transform/health

Stable Diffusion 서버 연결 상태를 확인합니다.
